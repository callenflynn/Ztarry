# Ztarry

Ztarry is a terminal-based Windows resource monitor

# Installation

## Installer

download https://github.com/callenflynn/Ztarry/releases/latest/download/installer.exe and run it for a silent install. Ztarry will be added to PATH

## CPKG Package Manager

run ``` cpkg install ztarry.callenflynn ``` 

## Portable

download https://github.com/callenflynn/Ztarry/releases/latest/download/Ztarry.exe for the latest portable build
